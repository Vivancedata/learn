# SQL for Data Analysis

SQL (Structured Query Language) is the language of data. Every data role requires SQL proficiency. This course takes you from basics to advanced analysis techniques.

## Learning Outcomes

- Write SQL queries to retrieve and filter data
- Use aggregations to summarize data (COUNT, SUM, AVG, etc.)
- Combine data from multiple tables with JOINs
- Create subqueries and CTEs for complex analysis
- Use window functions for running totals and rankings
- Optimize query performance
- Apply SQL to real business analytics problems
- Work with dates, strings, and NULL values effectively

## Prerequisites

- Basic computer skills
- Understanding of spreadsheets (Excel, Google Sheets)
- No prior programming experience needed
- Interest in working with data

## Course Structure

### Section 1: SQL Fundamentals
SELECT statements, filtering with WHERE, sorting, and basic operations.

### Section 2: Aggregation and Grouping
COUNT, SUM, AVG, MIN, MAX, GROUP BY, and HAVING clauses.

### Section 3: Joining Tables
INNER JOIN, LEFT JOIN, RIGHT JOIN, FULL OUTER JOIN, and self-joins.

### Section 4: Advanced Queries
Subqueries, CTEs, CASE statements, and set operations.

### Section 5: Window Functions
ROW_NUMBER, RANK, running totals, moving averages, and LAG/LEAD.

### Section 6: Real-World Analytics Projects
Apply SQL to sales analysis, customer segmentation, and funnel analysis.

## Why SQL for Data Analysis?

**Universal Skill**: Used in every company with data

**High Demand**: Required for data analyst, data scientist, and BI roles

**Immediate Impact**: Query databases directly for instant insights

**Foundation**: Required before learning advanced tools like Python/R

**Portable**: Works across PostgreSQL, MySQL, SQLite, BigQuery, Snowflake
